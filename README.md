# jsonify

![carbon](https://user-images.githubusercontent.com/3372598/175796597-c4baa723-3af7-446a-823c-58ec9dfea468.png)

Convert Javascript Object literals to JSON

Single quotes?
Unquoted properties?
Trailing commas?
Comments?

Just turn this:
```js
/* Javascript object */
{
  foo: 'bar', // comment
  'hello': `world`, // trailing comma?
}
```

Into this:
```json
{
    "foo": "bar",
    "hello": "world"
}
```
